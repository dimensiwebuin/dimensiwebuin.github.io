Apa Itu Dimensi Web?

Dimensi Web merupakan salah satu Badan Semi Otonom (BSO) HIMATIF UIN Sunan Gunung Djati Bandung yang bertujuan untuk menghimpun, meningkatkan, serta memanfaatkan potensi yang dimiliki mahasiswa Teknik Informatika UIN Sunan Gunung Djati Bandung dalam bidang pengembangan aplikasi berbasis web.

Our Vision

Visi

Menjadikan Dimensi Web sebagai sarana pengembangan potensi mahasiswa teknik informatika Universitas Islam Negeri Sunan Gunung Djati Bandung dalam bidang pembuatan aplikasi berbasis web agar terciptanya sumber daya manusia berkualitas yang memiliki jiwa kreatif dan inovatif dengan dilandasi iman dan taqwa kepada Tuhan Yang Maha Esa

Misi

  • Menciptakan lingkungan komunitas yang mempunyai rasa kekeluargaan dan solidaritas yang kuat.
  • Menciptakan sumber daya yang berkualitas.
  • Menjaga dan meningkatkan nama baik Jurusan Teknik Informatika Universitas Islam Negeri Sunan Gunung Djati Bandung, HIMATIF dan BSO Dimensi WEB

What We Do?

Front End

Seorang Front-End Web Developer bertanggung jawab untuk merancang dan membangun berbagai aplikasi web secara responsif, interaktif, dan juga user friendly.

Back End

Back-End Developer bertanggung jawab dalam mengelola server, aplikasi, dan database agar dapat saling berkomunikasi dengan baik dan lancar.

Testing

Testing software bertujuuan untuk memastikan software terhindar dari bug dan memiliki kualitas yang baik

Soft Skill

Selain berdiskusi mengenai hard skill, kita akan berdiskusi juga mengenai soft skill agar timbul jiwa profesionalitas para member.
